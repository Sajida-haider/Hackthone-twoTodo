# Next.js Frontend Development Skill

## Purpose
This skill provides expertise in Next.js frontend development, including App Router, component development, styling with Tailwind CSS, and integration with backend services.

## Capabilities
- Create Next.js components using modern patterns
- Implement App Router structure (app directory)
- Integrate with various authentication systems
- Develop responsive UI with Tailwind CSS
- Handle API routes and client-server communication
- Implement forms and user interactions
- Manage state and data fetching patterns
- Optimize for performance and SEO

## Best Practices
- Use the App Router for new projects
- Follow Next.js conventions for file structure
- Implement proper TypeScript typing
- Use server and client components appropriately
- Leverage Next.js built-in optimizations
- Follow accessibility guidelines
- Implement proper error handling and loading states

## Common Patterns
- Layout and template components
- Route handlers (API routes in app directory)
- Client components with 'use client' directive
- Server components by default
- Loading and error boundaries
- Form handling with React Hook Form or similar
- Data fetching with React Query or SWR

## Integration Points
- Backend API integration
- Authentication providers
- Database connections
- Third-party services
- Environment configuration
- Build and deployment optimization
